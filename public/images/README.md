# Image Placeholder

This folder should contain the following images:

## Required Images:

1. **hero.jpg** (500x500px or square ratio recommended)
   - Professional portrait for hero section
   - Professional headshot or high-quality photo
   - Used in the main hero section (right column on desktop)

2. **about.jpg** (500x500px or square ratio recommended)
   - Professional or lifestyle image for about section
   - Can be same photo as hero or different image
   - Used in the about section (left column on desktop)

## How to Add Images:

1. Save your image files as .jpg, .png, or .webp format
2. Place them in this `public/images/` directory
3. The website will automatically display them

## Image Requirements:

- Format: JPG, PNG, or WebP (JPG recommended for photos)
- Size: 500x500px minimum (or similar aspect ratio)
- Quality: High-quality, professional photos
- Color: Should work with the dark luxury theme
- File names: Keep as `hero.jpg` and `about.jpg` (or match filenames in components)

## Note:

Until real images are added, the website displays elegant SVG placeholder cards with instructions.

The placeholder is a graceful fallback that keeps the design looking premium while the real images are being prepared.
